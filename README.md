# Energy-Consumption-Prediction-using-Linear-Regression
This project applies Machine Learning Regression techniques to predict Energy Consumption based on different household and environmental factors. The implementation is built completely from scratch using Python, NumPy, Pandas, and Matplotlib without using machine learning libraries such as Scikit-learn.
# Energy Consumption Prediction using Linear Regression

## Overview

This project implements Simple and Multiple Linear Regression from scratch using Python and Gradient Descent optimization to predict energy consumption.

The project analyzes how different features affect energy usage and compares model performance using different learning rates and epochs.

---

## Dataset Features

- Square Footage
- Number of Occupants
- Appliances Used
- Average Temperature

Target:
- Energy Consumption

---

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib

---

## Implemented Algorithms

### Simple Linear Regression
Implemented using:
- Gradient Descent
- Mean Squared Error (MSE)

### Multiple Linear Regression
Implemented using:
- Vectorized Gradient Descent
- Multiple Features

## Results

The project generates:

## How to Run

1. Install required libraries:

```bash
pip install numpy pandas matplotlib
```

2. Place the dataset file:

```bash
assignment1dataset.csv
```

3. Run the script:

```bash
python main.py
```
